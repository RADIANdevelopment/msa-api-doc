### Получение списка групп операций

Для получения списка групп операций необходимо отправить **`GET`** запрос на **`/api/operation_group`**<br>

Пример ответа:<br>
```
[
    {
        "_id": "5cbc39755e42968e114d6318",
        "name": "Производственные рабочие места"
    },
    {
        "_id": "5ceee89502736acec4e97f24",
        "name": "Отдел контроля качества"
    }
]
```

<br>

### Создание группы для операций

Для создания группы для операций необходимо отправить **`POST`** запрос на **`/api/operation_group`**<br>

Пример отправки параметров:<br>
```
{
    "name": "Аутсорс"
}
```
Параметры:<br>
**name*** - Имя группы<br>

Ответ:<br> коды состояний 204 или 404.

<br>

### Изменение названия группы для операций

Для изменения названия группы для операций необходимо отправить **`PUT`** запрос на **`/api/operation_group`**<br>

Пример отправки параметров:<br>
```
{
    "_id": "5ceef0bee3c8c442a704b03a",
    "name": "Аутсорс"
}
```

Параметры:<br>
**_id*** - Идентификатор изменяемой группы<br>
**name*** - Новое имя группы<br>

Ответ:<br> коды состояний 204 или 404.

<br>

### Удаление группы для операций

Для удаления группы для операций необходимо отправить **`DELETE`** запрос на **`/api/operation_group/:id`**<br>

Параметры:<br>
**id*** - Идентификатор удаляемой группы<br>

Ответ:<br> коды состояний 204 или 404.

<br>

### Получение списка операций для группы

Для получения списка клиентов необходимо отправить **`GET`** запрос на **`/api/operation/:og_id`**<br>

Пример ответа:<br>
```
[
    {
        "_id": "5cbc39755e42968e114d6317",
        "name": "Cшивание комплекта",
        "description": "Подбор лекал, по данным из заказа и раскройка материалов"
    },
    {
        "_id": "5cbc3c325e42968e114d631d",
        "name": "Отстрочка",
        "description": "Подбор лекал, по данным из заказа и раскройка материалов"
    }
]
```

Параметры:<br>
**og_id*** - Идентификатор группы<br>

<br>

### Создание операции

Для создания операции необходимо отправить **`POST`** запрос на **`/api/operation`**<br>

Пример отправки параметров:<br>
```
{
	"_id": "5d5a556d52aa910e046fc9dc",
	"og_id": "5cbc39755e42968e114d6318",
	"name": "Упаковка",
	"description": "Упаковка",
	"sort": 
        {
        "sort\uFF0EМарка": 1,
        "sort\uFF0EПоколение вашего автомобиля": 1
        }
}
```

Параметры:<br>
**og_id*** - Идентификатор группы новой операции<br>
**name*** - Имя операции<br>
**description** - Описание операции<br>
**sort** - Сортировка для операции<br>
**deskbook*** - id справочника<br>

Ответ:<br> коды состояний 204 или 404.

<br>

### Редактирование операции

Для редактирования операции необходимо отправить **`PUT`** запрос на **`/api/operation`**<br>

Пример отправки параметров:<br>
```
{
	"_id": "5d5a556d52aa910e046fc9dc",
	"og_id": "5cbc39755e42968e114d6318",
	"name": "Упаковка",
	"description": "Упаковка",
	"sort": 
        {
        "sort\uFF0EМарка": 1,
        "sort\uFF0EПоколение вашего автомобиля": 1
        }
}
```

Параметры:<br>
**og_id*** - Идентификатор группы новой операции<br>
**name*** - Имя операции<br>
**description** - Описание операции<br>
**sort** - Сортировка для операции. Если сортировка по времени поступления, то отправляем "sort":[]<br>
**deskbook*** - id справочника<br>

Ответ:<br> коды состояний 204 или 404.

<br>
